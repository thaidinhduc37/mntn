# Trường Mầm non Tuổi Ngọc - Website

Website chính thức của Trường Mầm non Tuổi Ngọc, xã Quảng Phú, tỉnh Đắk Lắk.

## Mô tả

Website được xây dựng để cung cấp thông tin về trường, chương trình giáo dục, tin tức và hoạt động của trường mầm non. Website có thiết kế hiện đại, responsive và thân thiện với người dùng.

## Tính năng

- **Trang chủ**: Giới thiệu tổng quan về trường
- **Slider Hero**: Carousel hình ảnh với thông tin nổi bật
- **Thống kê**: Hiển thị số liệu về học sinh, giáo viên, lớp học
- **Giới thiệu**: Tầm nhìn, sứ mệnh và giá trị cốt lõi
- **Chương trình giáo dục**: Các chương trình cho từng độ tuổi
- **Tin tức**: Thông báo và hoạt động của trường
- **Thư viện hình ảnh**: Gallery với bộ lọc và lightbox
- **Liên hệ**: Form liên hệ và thông tin liên lạc
- **Responsive Design**: Tối ưu cho mọi thiết bị
- **Accessibility**: Hỗ trợ người khuyết tật

## Công nghệ sử dụng

- **HTML5**: Cấu trúc semantic
- **CSS3**: Styling hiện đại với CSS Grid, Flexbox
- **JavaScript**: Tương tác và animations
- **Font Awesome**: Icons
- **Google Fonts**: Typography (Inter font)

## Deploy lên GitHub Pages

1. **Tạo repository** trên GitHub
2. **Upload** toàn bộ source code
3. **Bật GitHub Pages** trong Settings
4. **Chọn branch** main/master
5. Website sẽ có địa chỉ: `https://username.github.io/repository-name`

## Hình ảnh cần thiết

Các file hình ảnh cần được thêm vào thư mục `assets/images/`:

### Logo và Brand

- `logo.png` - Logo trường (60x60px)

### Hero Slider

- `hero-1.jpg` - Hình ảnh trường (1920x1080px)
- `hero-2.jpg` - Hoạt động học tập (1920x1080px)
